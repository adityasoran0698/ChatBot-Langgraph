# 🤖 Chatbot Using LangGraph (MERN + GenAI)

A full-stack AI chatbot application built using **LangGraph**, **Node.js**, and **React (Vite)**.
This project demonstrates how to build an intelligent conversational system using modern **Generative AI + Full Stack Development**.

---

## 🚀 Features

* 💬 Interactive Chat UI (React)
* 🧠 AI-powered responses using LangGraph
* 🔐 Secure API handling with environment variables
* ⚡ Fast frontend using Vite
* 🔄 Real-time communication between frontend & backend
* 🧩 Modular and scalable architecture

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/chatbot-using-langgraph.git
cd chatbot-using-langgraph
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
```

Create a `.env` file in backend:

```env
OPENAI_API_KEY=your_api_key_here
PORT=5000
```

Run backend:

```bash
npm start
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```
